while반복문: 시간을 기반으로 반복하기

시간을 기반으로 반복하는 예를 살펴보자. 시간을 기반으로 반복하려면 '유닉스 타임'
이라는 개념을 알아야 한다. '유닉스 타임(Unix Time)이란 세계 표준시(UTC)로, 1970
년 1월 1일 0시 0분 0초를 기준으로 몇 초가 지났는지를 정수로 나타낸 것을 말한다.
파이썬에서 유닉스 타임을 구할때는 다음과 같은 코드를 사용한다.

먼저 시간과 관련된 기능을 가져온다.
>>> import time
그런 다음 유닉스 타임을 구한다.
>>> time.time()
1557241486.6654928

# 시간과 관련된 기능을 가져옵니다.
import time

# 변수를 선언합니다.
number = 0

# 5초 동안 반복합니다.
target_tick = time.time() + 5
while time.time() < target_tick:
    number += 1

# 출력합니다.
print("5초 동안 {}번 반복했습니다.".format(number))

실행결과
5초 동안 14223967번 반복했습니다.
